# Section 233 — Using evidence known to be false.

**Act:** Bharatiya Nyaya Sanhita, 2023  
**Chapter:** 14 — Of False Evidence And Offences Against Public Justice

---

Whoever corruptly uses or attempts to use as true or genuine evidence any evidence which he knows to be false or fabricated, shall be punished in the same manner as if he gave or fabricated false evidence. 
 BNSS Classification 
 The same as for giving or fabricating false evidence. 
 Non-cognizable 
 According as offence of giving such evidence is bailable or nonbailable. 
 Triable by Court by which offence of giving or fabricating false evidence is triable.
